How It Works

Timeline B allows users to generate narrative scenarios based on counter-factual historical events. Our system automates the creation of visuals, audio, and text, assembling them into short-form content.

1. Generate

AI creates alternative history scripts and imagery based on user prompts.

2. Assemble

Media is compiled into 9:16 MP4 format suitable for mobile viewing.

3. Publish

Content is uploaded directly to your TikTok account with AI disclosures.

Platform Compliance & Technical Data

Timeline B integrates with TikTok's Content Posting API strictly adhering to developer policies. Below is an overview of our integration scope for audit verification.

Scopes & Permissions

  • video.publish: Used to publish videos to the authenticated user's account.
  • video.upload: Used to upload the video binary file (MP4).

Publishing Flow

  1. Backend generation of images/audio/text.
  2. Assembly into MP4 (H.264, 9:16 aspect ratio).
  3. API Call: /v2/post/publish/video/init/
  4. Compliance: The parameter ai_generated=true is enforced on all uploads to disclose AI nature.
  5. Video upload via returned URL.
  6. Status check via /v2/post/publish/status/fetch/.
